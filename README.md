<p>ncov19-cli is a 2019 Novel Coronavirus COVID-19 (2019-nCoV) CLI based tracker</p>

# Installation

```
$ npm install -g ncov19-cli
```

# Usage

Get summary of ncov19-cli cases

```
$ ncov19
```

Get by location

```
$ ncov19 --location=Philippines
```

or

```
$ ncov19 -l Philippines
```
